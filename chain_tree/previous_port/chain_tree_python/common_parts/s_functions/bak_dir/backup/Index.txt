TEMPLATE CONTENTS SUMMARY
==========================

The workflow_templates.mako file contains:

1. retry_block(branch_id, max_retries)
   └─ Implements retry logic with configurable attempts
   
2. timeout_block(branch_id, timeout_seconds)
   └─ Implements timeout checking
   
3. error_handler(branch_id, error_message)
   └─ Handles errors with logging and notifications
   
4. sequential_steps(*step_names)
   └─ Executes multiple steps in sequence


QUICK START GUIDE
=================

1. Import the class:
   from lisp_sequencer import LispSequencer

2. Create your executor function:
   def my_run(handle, ast):
       # Your code here
       return result

3. Initialize:
   sequencer = LispSequencer(
       handle={"id": 123},
       run_function=my_run,
       user_macro_file='workflow_templates.mako'
   )

4. Execute with templates:
   instruction = """
   (pipeline
       <%retry_block(0, 3)%>
       (@DO_WORK)
       'CFL_FUNCTION_TERMINATE)
   """
   result = sequencer.run(instruction)

COMMON ISSUES
=============

Issue: "workflow_templates.mako shows blank or weird characters"
Solution: The file uses <% %> syntax which browsers interpret as HTML
         → Use workflow_templates.html instead for viewing
         → Or use workflow_templates_explained.txt

Issue: "Can't see the template code"
Solution: Open workflow_templates_raw.txt for plain text version

Issue: "Want to modify templates"
Solution: Copy content from workflow_templates_raw.txt,
         edit it, and save as YourName.mako
